# 🖥️ Google Ads SERP Simulator (2026 Updated)

A pixel-perfect Google Search Engine Results Page (SERP) emulator. Designed for media buyers and SEOs to validate character limits, truncation, and visual impact before pushing campaigns live in Google Ads Editor.

### 🚀 Live Demo & Web Tool
**Test your ad copy live here:** 👉 [Google Ads SERP Simulator by David Tamachi](https://davidtamachi.ca/google-serp-simulator.html)

### Key Features
* **Dynamic Truncation:** Accurately mimics Google's pixel-width truncation (not just character counts) for Title Tags and Ad Headlines.
* **Mobile vs. Desktop Views:** Toggle between viewport environments to see how ad extensions render.
* **Dark Mode Native:** Beautiful UI built for modern marketers.

### Use Cases
* Validating Responsive Search Ads (RSAs) offline.
* Client presentations and ad mockup approvals.
* Checking meta title and description pixel limits for organic SEO.
